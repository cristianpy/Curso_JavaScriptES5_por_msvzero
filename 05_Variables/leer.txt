Para que sirven las variables?
------------------------------
Las variables son utilizadas para almacenar valores de datos.

Que tipos de valores pueden almacenar las variables en javascript?
-------------------------------------------------------------------
Los datos pueden tener valores de los siguientes tipos:

Primitivos
-----------------
Representan datos simples.

    * boolean: true o false
    * null : valo nulo
    * undefined: Una variable a la que no se le ha asignado un valor tiene el valor indefinido
    * Number: 1, 2, 3 ....
    * String: "Juan Perez"

 No Primitivos
 --------------
 Representan datos complejos.
    *Objetos

Operadores  
-----------
Fuente: https://www.w3schools.com/js/js_assignment.asp

Aritmeticos
--------------
Operator	Description
+	        Addition
-	        Subtraction
*	        Multiplication
/	        Division
%	        Modulus (Remainder)
++	        Increment
--	        Decrement


Asignacion
------------
Operator	Ejemplo	    Equivalente
=	        x = y	    x = y
+=	        x += y	    x = x + y
-=	        x -= y	    x = x - y
*=	        x *= y	    x = x * y
/=	        x /= y	    x = x / y
%=	        x %= y	    x = x % y


Comparacion
----------------
Operator	Description
==	        igual a
===	        igual en valor y tipo
!=	        no es igual en valor
!==	        no es igual en valor y tipo
>	        mayor que
<	        menor que
>=	        mayor o igual que
<=	        menor o igual que
?	        operador ternario


Logicos
-------
